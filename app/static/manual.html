Manual breve de TrazoCad v69

OCR dirigido y reconstrucción documental.

Manual

Esta versión prioriza cuatro cosas: fidelidad visual del plano completo, mejor preservación del rótulo, OCR por regiones para cotas/textos/rótulos y un DXF con capas separadas para geometría y textos cuando el OCR logra leer contenido.

Flujo

1. Subí una imagen PNG/JPG/BMP/TIFF. 2. Elegí tipo de dibujo, hoja y orientación. 3. Usá la ayuda guiada para indicar prioridad de fidelidad, rótulo, cotas o geometría. 4. Procesá y revisá PDF, JPG/PNG y DXF.

Notas guiadas

La temática y la acción sugerida sí se transforman en instrucciones internas. El texto libre sirve para pedir un refuerzo puntual, por ejemplo conservar rótulo completo o reforzar líneas punteadas.

Qué mejora esta versión

El PDF ocupa mejor la hoja, el JPG/PNG salen desde una base de presentación más fiel, el DXF de nube de puntos usa una base raster más completa y el OCR por regiones intenta recuperar cotas, textos y rótulos con prioridad al pie del plano.

Límites actuales

Los textos todavía no quedan garantizados como contenido perfecto ni completamente editable. En planos difíciles conviene validar el resultado y, si hace falta, pedir otra pasada con una instrucción guiada distinta.
